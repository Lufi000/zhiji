## 4. 描述（Description，4000 字符内）

**推荐版本（约 600 字）：**

知几是一款八字（四柱）排盘与命盘解读 App，希望陪你轻松得到一张清晰、可查证的命盘，从传统框架里多了解自己一点、多看见一点人生的节奏。

【轻轻排盘】
· 输入出生日期、时间与性别，就能得到年柱、月柱、日柱、时柱，以及大运、流年。
· 规则是透明的：节气、夜子时、节令年等都有据可依，方便自己对照，或给懂的人一起看。

【命盘慢慢看】
· 日主卡片：日干、五行与简短意象，用有机图形安静呈现。
· 四柱：每柱都有天干地支、十神、藏干；想多看一点，可以展开「能量互动」，看看合会、冲刑害等五行关系。
· 大运与流年：起运年龄、每步大运十年、流年列表；选某一步大运、某一年，就能看到它与命盘之间的能量互动。

【能量互动】
· 合会（天干五合、地支六合 / 三合 / 半合）、冲刑害、枯、贪生忘克等关系都会温和标注，合化与半合五行也一并呈现。

【复制到 AI】
· 命盘页有一键「复制到 AI」，会把格式化好的命盘文本复制到剪贴板，你可以粘贴到 DeepSeek 等 AI 里，请它帮忙多解读一点、或一起想想规划，让传统智慧与当下生活轻轻连在一起。

【我们的想法】
· 不强调算命或吉凶预测，更在意自我认知与人生节奏；体验上尽量克制、不制造焦虑。
· 静时厚积，风起乘势。

## 9. 隐私政策与支持 URL（必填）

上架前需准备：

| 项目 | 说明 |
|------|------|
| **隐私政策 URL** | 需单独撰写并托管（如 GitHub Pages、自建站、Notion 公开页等），将链接填入 App Store Connect。此前项目内的 `privacy-policy.html` 已删除，需重新创建并部署。 |
| **支持 URL** | 用户获取帮助的页面（如使用说明、反馈方式、邮箱）。同上，需重新创建并托管后填入。 |

两处 URL 在 App Store Connect → 你的 App → App 信息 / 版本信息 中填写。
